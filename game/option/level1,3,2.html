Question: What does HTML stand for?

  1. Hyper Text Markup Language
  2. High Transfer Machine Language
  3. Hyperlink Transfer Markup Language

Correct Answer: 1. Hyper Text Markup Language

Reason: HTML stands for Hyper Text Markup Language. It is the standard language used to create and design the structure of webpages. The other options are incorrect as they do not accurately represent HTML.
